# InvestSmart AI

AI-powered stock research, portfolio analysis, and financial news simplification — built for investors.

## Tech Stack

- **Next.js 15** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS**
- **Recharts** (portfolio charts)
- **Lucide React** (icons)

## Installation

```bash
cd investsmart-ai
npm install
```

## Run (Development)

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Run (Production)

```bash
npm run build
npm start
```

## Pages

| Route | Description |
|-------|-------------|
| `/` | Landing page with hero, features, testimonials |
| `/research` | AI stock research — AAPL, MSFT, NVDA, TSLA |
| `/portfolio` | Portfolio analyzer with pie charts and scoring |
| `/news` | Financial news simplifier with structured analysis |

## Notes

- All data is mocked — no backend or API calls required
- Stocks supported in research: AAPL, MSFT, NVDA, TSLA
- Nothing constitutes financial advice
